## About the project

my site on nextjs

Tools

- Lang - `TypeScript`
- Framework - `NextJs`
- For multilingual - `i18n`
- For CI/CD - `GitHub Actions (self-hosted ubuntu)`
- Reverse Proxy / ssl - `Nginx & brotli`
- Code analysis - `eslint`

- Start `npm run dev`
